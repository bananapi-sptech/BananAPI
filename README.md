### BananAPI

Repositório responsável pela **BananAPI**, um sistema desenvolvido para monitoramento de temperatura em câmaras frias utilizadas em entrepostos de maturação de bananas.

A **BananAPI** é responsável por gerenciar a comunicação entre os sensores de temperatura, o banco de dados e as aplicações que exibem essas informações ao usuário. Através dela, é possível registrar leituras de temperatura, armazenar os dados de forma organizada e disponibilizá-los para consulta e análise.

#### Principais funções
- Receber e registrar leituras de temperatura enviadas pelos sensores.
- Armazenar os dados em banco de dados.
- Disponibilizar endpoints para consulta das medições.
- Permitir a integração com o sistema web do BananAPI.

#### Objetivo
Garantir o armazenamento e a disponibilidade dos dados de temperatura das câmaras frias, auxiliando no controle do processo de maturação das bananas e contribuindo para a qualidade do produto final.
